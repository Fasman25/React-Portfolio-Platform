## Personal Project Showcase App
  Tools Used :

React (v18+)

Vite (Building Tool)

CSS3 (Custom Responsive Design)

Jest & React Testing Library (Unit Testing)

  App Features
Dynamic Project List: Displays a curated gallery of agency work.

Search & Filter: Real-time project filtering to help clients find specific types of work instantly.

Project Management: An integrated form that allows the agency to add new projects dynamically without touching the code.

Responsive UI: Optimized for mobile, tablet, and desktop viewing, following a modern card-based design mockup.

  Setting up :
Follow these steps to get the project running locally on your machine.

1. Prerequisites
Ensure you have Node.js installed (v16 or higher recommended).

2. Installation
Clone the repository and install the dependencies:

Bash
git clone <your-repository-url>
cd portfolioPlatform
npm install
3. Running the App
Start the development server:

Bash
npm run dev
Open http://localhost:5173 in your browser to view the application.

4. Running Tests
To ensure the application's components are functioning correctly:

Bash
npm test
